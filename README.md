# 🔴 Manchester United AI News Agent

맨체스터 유나이티드 관련 최신 뉴스를 자동으로 수집하고  
한국어로 요약하여 디스코드로 전송하는 AI 에이전트입니다.

## 📌 Features

- Google News RSS 기반 뉴스 수집
- OpenAI API를 활용한 뉴스 요약
- 팬 관점 코멘트 생성
- 디스코드 자동 전송
